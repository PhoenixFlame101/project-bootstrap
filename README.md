# Project Bootstrap

Adds a .gitignore file for the language specified in the CLI argument.

## Installation

    echo "GITHUB_TOKEN=<YOUR AUTH TOKEN>" > .env
    cargo install --path .

Syntax

    project-bootstrap <language>
